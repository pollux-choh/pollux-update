# Pollux Update Metadata

이 repository는 Pollux 애플리케이션들의 업데이트 메타데이터를 GitHub Pages로 제공합니다.

## 구조

각 애플리케이션별로 디렉토리가 있습니다:
- `pollux-markdown/` - POLLUX MARKDOWN 업데이트 정보
- `pollux-ergon/` - POLLUX ERGON 업데이트 정보
- `pollux-vive/` - POLLUX VIVE 업데이트 정보

## GitHub Pages

이 repository는 GitHub Pages를 통해 정적 파일을 제공합니다:
- Base URL: `https://pollux-choh.github.io/pollux-update/`
- 각 애플리케이션의 `latest.json` 파일이 최신 버전 정보를 제공합니다.

## 자동 업데이트

빌드 워크플로우에서 자동으로 메타데이터가 업데이트됩니다.
